## Online Electronic Product Mobile Phone Mall

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4ffb845b0ed144ab8b9871cf257dd710/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Product Search and Selection
5. Product Preview
6. Shopping Cart
7. Order Management
8. Personal Information
9. Secure Logout

**Administrator Functions:**
1. Login
2. Homepage Data Statistics
3. User Management
4. Role Management
5. Product Category Management
6. Product Management
7. Order Management
8. Secure Logout
